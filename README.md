# 数据可视化交互平台

一个功能丰富的数据可视化网站，支持多种图表类型和样式自定义，基于 ECharts 构建。

## 🌟 功能特性

### 图表类型
- **折线图** - 支持平滑曲线显示
- **柱状图** - 多系列对比展示
- **饼图** - 数据占比可视化
- **散点图** - 分布关系分析
- **雷达图** - 多维度数据比较

### 样式自定义
- **配色方案** - 5种预设配色（默认、暖色、冷色、柔和、鲜艳）
- **字体设置** - 支持多种字体选择和大小调整
- **线条粗细** - 可调节图表线条粗细
- **数据标签** - 显示/隐藏数据点标签
- **网格线** - 控制网格线显示

### 数据编辑功能
- **表格编辑** - 直接在表格中修改数据值
- **实时同步** - 修改数据后图表立即更新
- **动态行/列管理** - 支持添加/删除行和列
- **分类名称编辑** - 可修改数据分类名称
- **数据验证** - 自动验证输入的数据格式

### 交互功能
- **实时预览** - 所有设置即时生效
- **数据管理** - 支持自定义数据输入
- **随机数据** - 一键生成测试数据
- **图表导出** - 导出高清PNG图片
- **重置设置** - 快速恢复默认配置

## 💡 使用指南

### 基本操作
1. **选择图表类型** - 从下拉菜单选择需要的图表
2. **调整样式** - 使用控制面板自定义图表外观
3. **查看效果** - 所有修改实时反映在图表上

### 数据管理
```json
{
  "categories": ["一月", "二月", "三月"],
  "series": [
    {
      "name": "系列1",
      "data": [100, 200, 300],
      "type": "line"
    }
  ]
}
```

### 键盘快捷键
- `Ctrl + S` - 快速保存当前设置
- `Esc` - 重置到默认状态

## 🎨 技术栈

- **前端框架**: 原生JavaScript
- **图表库**: ECharts 5.4.3
- **样式**: 纯CSS3，支持响应式设计
- **数据格式**: JSON

## 🔧 自定义开发

### 添加新图表类型
在 `app.js` 中的 `getChartOptions()` 方法添加新的图表配置。

### 扩展配色方案
在 `getColorScheme()` 方法中添加新的配色数组。

### 数据源集成
支持从API接口获取数据，只需修改数据加载逻辑。
